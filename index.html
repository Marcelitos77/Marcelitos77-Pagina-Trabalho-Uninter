MINHAS ESPECIALIDADES.

HTML

Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

CSS

CSS significa Cascading Style Sheets (Folha de Estilo em Cascatas). Ele tem esse nome porque aplica-se as estruturas de “cima para baixo”, ou seja, por meio de regras com prioridades para seguir.

JAVASCRIPT

O JavaScript é amplamente categorizado como uma linguagem de criação de scripts ou uma linguagem interpretada. O código em JavaScript é interpretado, ou seja, é traduzido diretamente no código de linguagem de máquina subjacente por um mecanismo de JavaScript.
